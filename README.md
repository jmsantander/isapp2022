# ISAPP 2022

This is a series of jupyter notebooks for the ISAPP 2022 school at University of Paris-Saclay.

## Requirements

The following libraries are required for all of them to work correctly.

- numpy
- matplotlib
- pandas
- healpy
- astropy
- astroplan
- ligo.skymap
- urllib3 
- astroquery 
- tsp_solver2
- scipy 
- scikit-spatial
